🚀 Space Shooter Game

A simple and fun 2D Space Shooter game built using HTML, CSS, and JavaScript.

Features

- Smooth left and right spaceship movement
- Fire bullets to destroy enemy ships
- Random enemy spawning system
- Collision detection between bullets and enemies
- Score tracking system
- Game Over screen with Play Again button
- Mobile-friendly touch controls
- Space-themed background and visual effects

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)

How to Play

1. Move your spaceship using the Left and Right buttons.
2. Press the Fire button to shoot bullets.
3. Destroy enemy ships to increase your score.
4. Avoid colliding with enemy ships.
5. The game ends when an enemy ship hits your spaceship.

Project Structure

- "index.html" – Main game layout
- "style.css" – Game styling and UI
- "game.js" – Main game logic
- "input.js" – Player controls
- "render.js" – Rendering game objects

Future Improvements

- Sound effects
- Explosion animations
- Multiple enemy types
- Power-ups and upgrades
- High score saving system
- Difficulty levels

This project was created as a learning project to practice JavaScript game development concepts such as rendering, collision detection, game loops, and user interaction.
